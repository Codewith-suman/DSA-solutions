## 🏆 **Top Tips for Success**

1. **Master the Traces** - Spend 70% time on tracing, 30% on theory
2. **Understand Complexity** - Know WHY each algorithm is fast/slow
3. **Practice Coding** - Type code yourself, don't just read
4. **Compare Algorithms** - Know when to use which algorithm
5. **Time Management** - Quick Sort & Dijkstra = high value
6. **Partial Marks** - Correct approach gets marks even if trace incomplete
7. **Notation Matters** - Use proper pseudocode format in exam
8. **Answer Template** - Follow consistent structure for all answers
