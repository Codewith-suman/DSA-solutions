## 💡 **How to Read Trace Tables**

### **Example: Quick Sort Trace**

```
Array: [7, 2, 8, 1]
Pivot: 1 (last element)

  j=0: 7<1? NO  →  [7, 2, 8, 1]
  j=1: 2<1? NO  →  [7, 2, 8, 1]
  j=2: 8<1? NO  →  [7, 2, 8, 1]
  
Swap pivot: [1, 2, 8, 7]
Position of pivot: 0

MEANING:
- j moves through array
- We count which elements are smaller than pivot
- When found, we swap to left side
- This is how we partition!
```

**Pro Tip**: Draw this yourself on paper while reading - it sticks better!
